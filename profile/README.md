# Tamga

**Open-source digital security research, from Türkiye.**

Tamga, siber savunma araştırmaları, tehdit analizi, yapay zekâ güvenliği, kriptografi ve güvenli yazılım geliştirme üzerine kaynaklı, savunma odaklı açık kaynak içerik üreten bağımsız bir araştırma alanıdır.

Bir dernek, vakıf, resmî kurum veya şirket değil — GitHub üzerinden yürüyen, katkıya açık bir çalışma alanıdır.

### Aktif Repolar

| Repo | Odak |
|---|---|
| [`cyber-defense-atlas`](../cyber-defense-atlas) | Türkiye bağlamında güncel dijital riskler, sektör notları, savunma checklist'leri |
| [`malware-threat-analysis`](../malware-threat-analysis) | Malware davranış kalıpları, tehdit sınıfları, savunma/tespit odaklı analiz |
| `ai-safety-lab` | AI/agent güvenliği, prompt injection farkındalığı, güvenli AI ürün tasarımı *(planlanıyor)* |
| `crypto-privacy-lab` | Kriptografi, mahremiyet, güvenli ürün geliştirme *(planlanıyor)* |

### Nasıl Katkı Verilir

Tüm katkılar fork → Pull Request akışıyla alınır. Hiçbir dış katkıcıya doğrudan yazma yetkisi verilmez. Detaylar için `.github/CONTRIBUTING.md`.

### Ne Yapmıyoruz

Çalışan exploit/PoC kodu, malware örneği, C2 kurulum rehberi, jailbreak/bypass prompt'u veya gerçek hedef/kurban bilgisi paylaşmıyoruz. Amacımız saldırı öğretmek değil, savunma bilgisi üretmek.
