# Tamga — `.github`

Bu repo, Tamga organizasyonundaki tüm açık kaynak repoları için ortak topluluk sağlığı dosyalarını (community health files) barındırır: katkı rehberi, güvenlik politikası, davranış kuralları, issue/PR şablonları ve paylaşılan araştırma metodolojisi.

Buradaki dosyalar GitHub tarafından otomatik olarak organizasyondaki diğer tüm repolara varsayılan olarak uygulanır (bir repo kendi `CONTRIBUTING.md` veya `SECURITY.md` dosyasını tanımlamadığı sürece).

## Bu repoda ne var

- `CONTRIBUTING.md` — issue → assignment → PR → review → merge akışı, katkıcı güven kademeleri (Tier 1/2/3)
- `SECURITY.md` — sorumlu açıklama politikası
- `CODE_OF_CONDUCT.md` — davranış kuralları
- `PULL_REQUEST_TEMPLATE.md` — tüm PR'larda kullanılan ortak şablon
- `ISSUE_TEMPLATE/` — içerik önerisi ve düzeltme şablonları
- `methodology/` — Verified Facts/Analyst Assessment ayrımı, kaynak güvenilirliği modeli, güvenli dil ilkeleri, katkıcı hakları

## Tamga nedir, ne değildir

Tamga, Türkiye'den çıkan, bağımsız ve açık kaynak bir dijital güvenlik araştırma alanıdır. Resmî bir kurum, dernek, vakıf veya savunma sanayi şirketi değildir. GitHub ve resmi web sitesi üzerinden yürür, kaynaklı bilgi üretir, savunma odaklıdır.

**Slogan:** *Open-source digital security research, from Türkiye.*
